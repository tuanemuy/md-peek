# 動作確認計画 — Issue #67: ServerInstance.shutdown() にべき等性ガードを追加

**Issue:** #67
**作成日:** 2026-03-24

---

## 確認環境

- ローカル開発環境
- `pnpm dev` でサーバー起動

## 確認項目

### 1. 通常のシャットダウン

- **目的:** Ctrl+C で正常にシャットダウンすること
- **手順:**
  1. `pnpm dev` でサーバーを起動
  2. ブラウザでプレビューが表示されることを確認
  3. Ctrl+C を押す
- **期待結果:** 「Shutting down...」→「Server stopped. Bye!」と表示されて正常終了
- **確認ポイント:** エラーメッセージが出ないこと

### 2. Ctrl+C 連打

- **目的:** 二重シャットダウンが安全に処理されること
- **手順:**
  1. `pnpm dev` でサーバーを起動
  2. Ctrl+C を素早く2回押す
- **期待結果:** エラーなく正常終了
- **確認ポイント:** `ERR_SERVER_NOT_RUNNING` エラーが出ないこと

## エッジケース・異常系

### 1. SIGTERM 送信

- **目的:** SIGTERM シグナルでも正常シャットダウンすること
- **手順:**
  1. `pnpm dev` でサーバーを起動
  2. 別ターミナルから `kill <PID>` を送信
- **期待結果:** 正常にシャットダウン
